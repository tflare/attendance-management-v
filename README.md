# attendance-management-v

## Project setup
```
npm install
```
### Firebase設定
```
プロジェクトのルート.env.localを作成し
以下を設定すること

VUE_APP_APIKEY=
VUE_APP_AUTHDOMAIN=
VUE_APP_DATABASEURL=
VUE_APP_PROJECTID=
VUE_APP_STORAGEBUCKET=
VUE_APP_MESSAGINGSENDERID=
VUE_APP_APPID=
VUE_APP_MEASUREMENTID=
```
